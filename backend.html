Use ChatGPT to quickly generate SQL queries based on a provided Schema

Overview

SQL is awesome but most of us are pretty bad at it. Let's have ChatGPT do it for us. We will be using a website called sql-practice.com to test the capabilities of ChatGPT when it comes to SQL.

We will provide the schema for the database below. There are multiple tables in their text format. Follow the instructions to get ChatGPT ready to create SQL based on this schema.

Schema

patients: patient_id INT, first_name TEXT, last_name TEXT, gender CHAR(1), birth_date DATE, city TEXT, province_id CHAR(2), allergies TEXT, height INT, weight INT

admissions: patient_id INT, admission_date DATE, discharge_date DATE, diagnosis TEXT, attending_doctor_id INT

doctors: doctor_id INT, first_name TEXT, last_name TEXT, specialty TEXT

province_names: province_id CHAR(2), province_name TEXT

Step 1: Try Out sql-practice.com

For a few minutes, get yourself acquainted with the website. When you get into the website you will see a large text field in the center with some instructions. Attempt some of the easy questions and maybe some medium if you’re feeling up to it. Once you use the website for about 5 minutes, move on to the next step.

Step 2: Give ChatGPT Relevant Information

ChatGPT has no idea that we are going to want it to provide SQL. So, we have to tell it. In your prompt, start out by letting it know that we have a database with multiple tables in it and that we want it to provide SQL queries. Then, say “The Database Schema and its Tables are provided below:” followed by the database schema (copy and paste it in).

Now, lets check if ChatGPT understood the schema we provided. Give it the following prompt "Display the schema as tables"

Compare the output to the website and see if ChatGPT understands the data we provided.

Step 3: See How Far it Can Get

Now that ChatGPT has all of the relevant information, complete as many questions as you can with it. This should be as simple as copying the prompt from sql-practice.com and pasting it into the text box on ChatGPT.

Step 4: Analyze

  • How did ChatGPT do?
  • What do you think of your experience using this as a tool for SQL?
  • Is there a better or easier way to set up ChatGPT to get it to provide accurate SQL queries?
  • What are the limitations of ChatGPT in generating SQL queries?
  • How does ChatGPT compare to other tools or methods for generating SQL queries?
