Frontend

Use ChatGPT to quickly generate HTML and CSS prototypes

Overview

Provided is a set of requirements we want for a basic front-end single-page application called "CodeQuest." CodeQuest is an interactive and engaging platform for college software engineering students to participate in coding challenges, collaborate with peers, and track their progress.

However, we do not want you to create this all by yourself! We want you to use ChatGPT to do the above tasks!

Frontend System Description

Requirements for the CodeQuest application:

  • A responsive navigation bar with four menu items: Home, Challenges, Leaderboard, and Profile.
  • A hero section with a captivating tagline and a call-to-action button to start a coding challenge.
  • A Challenges section displaying a list of coding challenges, each with a brief description, difficulty level, and a button to attempt the challenge.
  • A Leaderboard section showcasing the top performers in the community, highlighting their scores and accomplishments.
  • A Profile section where users can view their progress, completed challenges, and badges earned.
  • A footer with social media icons, contact information, and copyright information.

Step 1: Navigate to ChatGPT and the Code Pen to display your created front-end

ChatGPT: chat.openai.com

Create a codepen account and navigate to a new “pen” (this is where you will put any html and css for the page): codepen.io

Step 2: What to prompt for?

Next is to start the prompting process with ChatGPT. You should explain each of the requirements listed above and describe what types of items you want created (ie. html, css, overall theming, etc). Refer to the below example prompt if you are lost on what to prompt for.

Experiment by requesting ChatGPT to use CSS Frameworks like Bootstrap, Bulma, Foundation, etc.

******NOTE*****

If at any point GPT does not finish what it was outputting, just type “Continue” to have it continue from where it left off.

Step 3: Paste into codepen.io

Once you receive the generated code from ChatGPT, copy and paste the HTML, CSS, and JavaScript (if applicable) into the appropriate sections of your codepen.

Step 4: Evaluate Front-end

Review the generated front-end to ensure it meets the requirements and looks visually appealing. Make any necessary adjustments to the code as needed.

Step 5: Iterate and continue to prompt

If the generated code doesn't meet your expectations or you want to make further improvements, go back to ChatGPT and provide additional prompts. Iterate on the process until you are satisfied with the final result.

Step 6: Analyze

  • How did ChatGPT do?
  • What do you think of your experience using this as a tool for prototyping?
  • Is there a better or easier way to set up ChatGPT to get it to provide appealing frontend designs?
  • What are the limitations of ChatGPT in generating frontend prototypes?
  • How does ChatGPT compare to other tools or methods for generating HTML and CSS?

Example Prompt

Here is an example prompt you can use:

"ChatGPT, I need help creating the home page for a front-end single-page application called CodeQuest. CodeQuest is an interactive and engaging platform for college software engineering students to participate in coding challenges, collaborate with peers, and track their progress. The home page should include a responsive navigation bar with four menu items: Home, Challenges, Leaderboard, and Profile; a hero section with a captivating tagline and a call-to-action button to start a coding challenge; and a footer with social media icons, contact information, and copyright details. Please provide HTML and CSS code for this home page design, making sure the overall theming is modern, engaging, and gamified for college students."
